스마트 모빌리티 판매/대여 업체를 전문으로한 ERP입니다
매장의 전반적인 관리를 문서화 하지 않고 데이터화 시켜 관리하게 쉽고 효과적인 매장 관리를 위해 매입, 매출, 재고, 직원 관리 등을 제공합니다

관리자로 로그인중 점주로 로그인중 직원으로 로그인중

보고서
정재욱

- 공통 : 로그인 화면, 로그인 인터셉터, 로그인 프로세스, 차트, 검색
- 계정과목관리 : 관리자 권한으로 로그인시 계정과목 등록, 수정, 삭제, 조회가 가능합니다
- 지출관리 : 관리자 권한으로 로그인시 각 매장의 지출을 관리할 수 있습니다
- 지출/매출 : 점주 권한으로 로그인시 자신의 매장의 지출과 매출 현황을 알 수 있으며 지출(급여, 공과금/기타) 등록, 수정, 삭제, 조회가 가능합니다
※ 지출과 관련된 항목들은 그래프로 현황를 보여줍니다 ※

- 개인 블로그 : codenbike.tistory.com

문영성

- 기본상품정보등록 : 관리자일때만 가능 일반 점주들은 접근하여 검색 매입등록가능
- 매입,재고 : 기본상품정보를 통해 매입등록,삭제,조회 가능 하고 현재상품상태로 재고상태확인가능
- 판매,대여 : 매입품목으로 등록,수정,삭제,조회 가능하고 매출입력가능 매출입력시 현상품상태 변환
- 매출 : 판매,대여 매출등록한 금액 조회,삭제,검색 가능하며 월별로 그래프현황 나타냄 로그인시 관리자가아닐경우 해당매장만 나타남
- 로그인할때 관리자가아닐시 로그인점주 매장별로만 보여지게 되어있고 관리자일때만 전체매장 확인가능합니다
※ 매출항목은 그래프로 현황를 보여줍니다 ※

주석호

- 계약관리 : 관리자일때만 사용 가능 계약단가표,년별로 계약건 수 조회,계약된 목록 조회
- 계약하기 : 점주 아이디와 관리자만 사용 가능, 점주 아이디당 계약 1회 가능,수정은 관리자만 가능 및 수정시 입력값으로 등록
- 계약금 납부 관리 : 관리자일때만 사용 가능 ,계약을 하면 납부관리는 자동으로 생성,계약금 수정,조회 가능
- 매장 관리 : 관리자일 때만 매장 등록,수정,삭제 가능, 매장 등록 시 매장계약 자동 생성 및 회원에 있는 계약코드 수정,POS 등록,수정,삭제,조회 가능
- 거래처 관리 : 관리자가 아닐경우 해당되는 매장의 거래처만 사용, 거래처 등록,수정,삭제,조회 사용 가능 및 직원은 수정,삭제 불가능
※ 계약관리 년 계약건 수는 그래프로 보여줍니다. ※

박건영

- 회원관리 : 회원가입화면, 관리자권한 전체회원목록 검색,수정,삭제 가능
- 직원관리 : 점주권한 사용가능, 점주가 직원등록, 점주매장의 직원목록 검색,수정,탈퇴 가능
- 근태관리 : 점주와 직원 권한 사용가능, 점주와 직원 로그인중 직원아이디와 비밀번호 확인후 출퇴근등록, 점주일때 매장직원전체근태목록,검색
- 대여고객관리 : 관리자, 점주와 직원 사용가능, 대여고객 목록에서 대여고객등록,검색,수정,반납사용가능, 대여고객등록에서 대여상품목록에서 선택후 대여고객정보입력후 등록
